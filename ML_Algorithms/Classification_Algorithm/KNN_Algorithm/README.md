# KNN — K-Nearest Neighbours

_Work in progress — code not added yet._

KNN is probably the most straightforward classification algorithm out there. It doesn't really "learn" anything. It just looks at the data you already have, finds the closest matches to whatever you're trying to predict, and takes a vote.

Say K=5. You give it a new passenger/person/flower — it finds the 5 most similar ones from the training data, checks what category they were, and picks whichever one shows up most. That's literally it.
