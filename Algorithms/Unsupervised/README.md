# Unsupervised Learning Algorithms

Unsupervised learning finds patterns in data without being told the right answer. There's no target column, no labels, no "correct" answer to train against. The algorithm looks at the raw data and finds structure on its own.

Think of it like this: supervised learning is a student studying from an answer key. Unsupervised learning is a student sorting a pile of coins into groups with no instructions — they figure out the groupings themselves.
